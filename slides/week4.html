Bootstrap

What is Bootstrap

  • A front-end framework.
  • First developed at twitter.
  • Provides a system of css classes, components, and examples.
  • Some javascript components and themes.

Why

  • Great example of CSS components
  • Quick, solid foundation
  • Consistent appearance
  • Opinionated CSS reset

Bootstrap 4

  • The latest version
  • Flexbox focused
  • Lots of sites using older versions

Utilities

Documentation

Small helper classes to change styles in small ways.

Color

Colors are defined by purpose, e.g. 'danger' instead of 'red'

Spacing

Spacing is set through classes for consistency, much like h1 tags.

Range from .25rem to 3rem.

Layout

Documentation

Systems to fundamentally define the page structure.

Containers

Everything in the grid goes in a container.

Can be fluid or fixed.

12 Column Grid

12 equal columns.

Each horizontal section is a row.

Can be defined for different breakpoints.

Media Object

Bootstrap implementation of the same thing we talked about last week.

Components

Documentation

Prebuilt blocks of code for common functional needs.
